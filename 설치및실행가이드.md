# 초등학생 학습 챗봇 - 설치 및 실행 가이드

## 🚀 프로젝트 생성 및 설정

### 1. Next.js 프로젝트 생성

```bash
npx create-next-app@15.5.7 elementary-chatbot-frontend --typescript --tailwind --app
cd elementary-chatbot-frontend
```

### 2. 필수 패키지 설치

```bash
npm install jotai@2.10.3
npm install framer-motion@11.11.17
npm install lucide-react@0.460.0
npm install react-textarea-autosize@8.5.4
```

### 3. 폴더 구조 생성

```bash
mkdir -p src/{components,store,types,utils}
```

### 4. 파일 생성

위의 Artifact에 있는 각 파일들을 해당 경로에 생성합니다:

-   `src/types/chat.ts`
-   `src/store/chatAtoms.ts`
-   `src/utils/api.ts`
-   `src/components/Header.tsx`
-   `src/components/ChatMessage.tsx`
-   `src/components/ChatInput.tsx`
-   `src/components/QuickButtons.tsx`
-   `src/components/TypingIndicator.tsx`
-   `src/app/page.tsx`
-   `src/app/layout.tsx`
-   `src/app/globals.css`
-   `tailwind.config.ts`
-   `next.config.ts`
-   `tsconfig.json`
-   `package.json`

### 5. 환경 변수 설정

```bash
echo "NEXT_PUBLIC_API_URL=http://localhost:5000" > .env.local
```

## 🎮 실행 방법

### 1단계: 백엔드 서버 실행

```bash
# 커스텀 모델 생성
ollama show llama3.1:8b --modelfile > Modelfile
ollama create elementary-kor-teacher -f Modelfile
# 터미널 1: Ollama 실행
ollama run elementary-kor-teacher

# 터미널 2: 백엔드 서버 실행
cd ../elementary-chatbot/backend
source venv/bin/activate
python server.py
```

### 2단계: 프론트엔드 실행

```bash
# 터미널 3: Next.js 개발 서버
cd elementary-chatbot-frontend
npm run dev
```

### 3단계: 브라우저 접속

```
http://localhost:3000
```

## 🔧 트러블슈팅

### 백엔드 연결 실패

```bash

```

```bash

npm run dev -- -p 3001
```

```bash
# node_modules 재설치
rm -rf node_modules package-lock.json
npm install
```
